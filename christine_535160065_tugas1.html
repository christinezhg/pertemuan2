1.1 Sejarah HTML

HTML adalah bahwa dasar untuk menampilkan halaman web pada web browser. HTML merupakan singkatan dari Hypertext Markup Language. yang pertama adalah bahasa markup(penanda) berbasis text atau bisa disebut juga formating language. HTML adalah bahasa pemograman. HTML bukanlah bahasa pemograman tetapi bahasa markup formating

HTML dibuat oleh Tim Berners-Lee ketika masih bekerja untuk CERN dan dipopulerkan oleh browser Mosaic. Selama awal tahun 1990 HTML mengalami perkembangan yang sangat pesat. Setiap pengembangan HTML pasti akan menambahkan kemampuan dan fasilitas yang lebih baik dan versi sebelumnya. Sebelum suatu HTML disahkan sebagai suatu dokumen HTML standar, ia harus disetujui dulu oleh W3C untuk dievaluasi secara ketat. Setiap terjadi perkembangan suatu versi HTML, maka mau tak mau browser pun harus memperbaiki dan agar bisa mendukung kode-kode HTML yang baru tersebut. Sebab jika tidak, browser tak akan bisa menampilkan HTML tersebut.

Saat ini HTML sudah mencapai versi 5 atau biasa disebut HTML5. Versi ini memiliki banyak fitur yang dapat digunakan dan memiliki beberapa tambahan penggunaan script pada editor.

HTML biasa ditulis pada sebuah text editor (dreamwaver,notepad++,notepad,phpstrom,brackets,netbean,dll). Ketik kutipan coding HTML diatas pada texteditor anda dan kemudian simpan dengan format extention .html yang menandakan bahwa file tersebut adalah file HTML.

Item yang paling penting dalam melakukan pemograman web adalah editor teks atau IDE yang mempu mendukung Anda dalam menulis kode.
